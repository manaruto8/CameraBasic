# camerabasic
基于谷歌文档简单实现的自定义相机Demo，包含CameraX、Camera2、Camera的使用。已经实现点击拍照、长按录制、摄像头切换、手动点击对焦等功能。解决了预览画面变形、拍照旋转的问题。
